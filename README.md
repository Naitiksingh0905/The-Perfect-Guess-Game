# The-Perfect-Guess-Game
This is a small game which is made by using Python programming language using a simple library of Python. In this game the user have to guess the number which is choosed by the computer/system.

# Library used
Random inbuild library used in this project through which the system choose a random number from 1 to 100 and the used have to guess it. 

- It also calculate the user's the number of attempts by which the user have guessed the number correctly.
